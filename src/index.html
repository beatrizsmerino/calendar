Calendars

Calendar 1

Calendar of the current year, that starts the week on Sunday and it has the current date selected. You can scroll horizontally to see the months and click on the button show today to return to the current date. The button show all months display one by one or all months at the same time. And with the select of the languages you can change the language of the name months and weeks.

Calendar 2

This calendar, unlike the other one, starts the week on Monday and its language is in Spanish. It has previous and next buttons to show the other months, this makes that the structure of the calendar does not generate the html of the whole year, but it does it month by month, so you can see the month of the following year. It shows by default the current date, but if you scroll through the months, there are 3 buttons as shortcuts to go back to the previous year, to the current date and to the next year.
